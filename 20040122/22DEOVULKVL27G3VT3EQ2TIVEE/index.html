花車巡遊：
廣果道夜睇巡遊10部花車閃閃發光

大年初一，晏晝睇怕大家都要四出拜年，𢭃完利是跟手打番16圈，食埋晚飯再直落……直落做乜好呢？梗係去參觀一年一度嘅花車巡遊喇！活動已經踏入第9屆，今年重破天荒喺晚上舉行，無數花車燈泡閃閃發亮，配合埋沿途色彩斑斕嘅霓虹燈，肯定靚絕！今晚廣東道肯定迫爆場，有心捧場唔該早啲霸定位。

記者：陳震霖　攝影：陳盛臣、伍慶泉
模特兒：Vivian、Kelly（SunSpot）

花車大晒冷

舊年有12部花車巡遊，今年只得10部，不過整體開支800萬，比舊年多200萬，資金更集中，貴精不貴多，大家落足眼力慢慢睇喇！

「國泰民安」拉大隊打頭炮
國泰航空公司派出「國泰民安」花車打頭炮，最過癮係2架Q版人肉飛機，飛機一對大眼睛，下面多對腳滿場飛，好惹笑！國泰係全晚最大陣仗嘅花車隊伍，由機師、空姐及其他機組人員組成嘅150人巡遊隊伍，沿途會向大家派發紀念品，大家唔好執輸呀。

威威獅令駕「海洋公園歡躍迎新禧」
花車名水蛇春咁長，車身有2條發光海豚，開晒燈，遠睇勁Sharp！由公園吉祥物威威獅令帶領，大年初一佢會企喺車頭，同大家揮手拜年，跟車尾有一群圍住美猴王出場嘅猴子猴孫，美猴王會邊巡遊邊耍棍，兼擺Pose俾大家影相。

「美猴王」超大面譜最應節
香港中旅（集團）有限公司嘅「美猴王」花車勁應節，車頭大大個美猴王京劇臉譜，車身有粒通透金珠，綻放燦爛光芒，美猴拼金珠，寓意香港呢顆東方之珠繼續璀璨，猴年百業興旺。

精采表演隊伍
除咗花車，重有26隊嚟自世界各地同本地嘅表演隊伍，精采表演肯定令廣東道氣氛熾熱，睇埋至好走呀。

爆笑銅樂隊
「澳洲手舞足蹈活力樂隊」係非一般銅樂隊，音樂演奏會加入惡作劇、搗蛋、娛樂元素，隊員會扮袋鼠跳、俾鱷魚咬，諧趣動作必令你捧腹爆笑，啱！年初一最緊要開心爆笑！

熱鬧腰鼓獻技
「中國延安市腰鼓秧歌演出隊」來頭唔小，3年間囊括內地「全市腰鼓秧歌比賽一等獎」、「西部民歌邀請賽金獎」等獎項，隊員一身富地方色彩嘅服飾，打住腰鼓載歌載舞，場面肯定熱鬧。

勁誇表演服飾
「加拿大加勒比海文化委員會」隊員有參加世界各地嘉年華經驗，表演糅合服飾、舞蹈、音樂、戲劇元素於一身，共分9段故事，齋睇隊員拖住件高過佢哋嘅誇張服飾已經拍案叫絕。

美少女啦啦隊
「英國倫敦元旦巡遊及愛丁堡復活節節慶表演隊」表演5個環節，除「昇天飛鷹啦啦隊」美少女大跳大叫外，重請嚟彼德摩爾先生共賀猴年，佢係英國「街頭公告員」，專責報道皇室成員結婚、生子等大事，喺倫敦勁響朵㗎。

邊食邊睇巡遊
想一路食飯一路睇花車？都唔一定要留喺屋企睇電視嘅，等小記推介3間食肆畀大家喇。

1.HardRockCafe
1樓窗口位落地玻璃對正廣東道，一邊歎住香脆炸薯皮（$78），一邊睇花車，唔使同人迫餐死。
地址：尖沙嘴廣東道30號新港中心地下及1樓（23751323）

2.美心快餐
嫌貴，經濟啲咪去美心快餐囉，一樣窗口位，不過窗口位有限，霸靚位唔該請早。
地址：尖沙嘴廣東道54至60號帝國大廈1樓全層（27356065）

3.上海坊
想食中餐？去上海坊整鍋雲吞雞（大$260，小$150）、四川擔擔麵（$28）亦得，訂定窗口位到時咪邊歎邊睇囉。
地址：尖沙嘴廣東道68至80號恆利商業大廈2樓（31136993）

霸靚位睇花車
想快人一步霸定靚位睇花車巡遊？留意以下靚位推介喇。

1.廣東道2路口位
由於巡遊路線先行廣東道後轉九龍公園徑，所以貫通二路嘅海防道同北京道位置最正，梗係喇，花車隊伍一兜入九龍公園徑，你就可以衝去海防道同北京道另一邊，第一時間守住靚位。

2.巡遊隊伍轉彎處
無㗎！有成10部花車一齊巡遊，預咗慢駛，轉彎位就更加要減速喇，你重唔大把時間影相咩！
