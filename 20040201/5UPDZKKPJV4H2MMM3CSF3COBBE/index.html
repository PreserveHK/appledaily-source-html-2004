澳網賽沙芬惡鬥費達拿爭霸

【綜合澳洲外電】剛登上世界第一的瑞士球手費達拿，下一個目標就是在周日進行的澳網賽男單決賽稱王，但他要首度捧起此獎盃並不容易，因其對手是愈戰愈勇的俄羅斯球手沙芬。（有線62台今日8:00am起直播）
二號種籽的費達拿是首個取得世界排名首位的瑞士男網手，雖然他現時勢盛，不過就直言未敢輕視第二度晉身決賽的沙芬。費達拿說：「沙芬具備成為冠軍的條件，我很高興他能捲土重來。」
沙芬去年備受傷患困擾，世界排名跌至八十六位；惟他在今屆表現出色，晉級之路先後淘汰頭號種籽、前世界第一盧迪克，以及衞冕的阿加斯。
在○二年決賽負瑞典祖軒遜的沙芬表示：「我希望今次可捧盃而回。費達拿是個天才，跟他交手時必須小心；但我現時信心十足，可為球迷帶來驚喜。」

中台女雙闖青少年決賽
另外，中國的孫勝男和中華台北的成容簡殺入青少年組女雙決賽，將與捷克的維迪索娃及捷禾高娃爭冠軍。
